
# IBT Python & Data Structures Practice

Welcome to the **IBT Python & Data Structures Practice** repository.

## 📖 About

This repository contains Python programming exercises and Data Structures practice completed during the IBT class. The goal is to strengthen programming fundamentals, problem-solving skills, and algorithmic thinking through hands-on coding.

## 📂 Contents

* Python Basics
* Variables and Data Types
* Conditional Statements
* Loops
* Functions
* Lists, Tuples, Sets, and Dictionaries
* String Manipulation
* File Handling
* Exception Handling
* Object-Oriented Programming (OOP)
* Data Structures

  * Arrays
  * Linked Lists
  * Stacks
  * Queues
  * Trees
  * Graphs
* Practice Problems
* Assignments

## 🎯 Objectives

* Learn Python from beginner to intermediate level.
* Understand core Data Structures and their applications.
* Improve coding and problem-solving skills.
* Practice writing clean and efficient Python code.
